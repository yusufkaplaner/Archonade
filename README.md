# 👁️🎙️ ARCONODE
**Siber-Akademik Mantık ve Öğrenme Motoru**

[![Live Demo](https://img.shields.io/badge/Live_Demo-Play_Now-00F0FF?style=for-the-badge)](https://app-bgfanvos62v5.appmedo.com)
[![Built with MeDo](https://img.shields.io/badge/Built_with-MeDo-blue?style=for-the-badge)](https://app-bgfanvos62v5.appmedo.com)

ARCONODE, karmaşık mühendislik algoritmalarını (örneğin Dijkstra grafikleri) ve matematik problemlerini görsel olarak analiz eden, ardından bunları adım adım, Türkçe sesli anlatımla çözen interaktif bir AI asistanıdır. Bursa Teknik Üniversitesi (BTÜ) siber-akademik vizyonuyla geliştirilmiştir.

## 🚀 Öne Çıkan Özellikler

- **Neural Vision (Çok Modlu Görme):** Sorunun veya elle çizilmiş bir grafiğin fotoğrafını yükleyin, ARCONODE pikselleri mantığa dönüştürsün.
- **Cinematic Narration (Sesli Mantık):** Web Speech API kullanılarak her çözüm adımı pedagojik bir Türkçe sesle anlatılır.
- **Dynamic HUD:** Çözüm simülasyonu akarken, arka plandaki değişkenleri (örn. mesafe tablosu) gerçek zamanlı güncelleyen şık bir arayüz.
- **KaTeX Entegrasyonu:** Karmaşık matematiksel formüller yapay zekadan ham haliyle alınır ve kusursuz bir şekilde ekrana renderlanır.

## 🛠️ Teknolojiler

- **Altyapı:** MeDo Full-Stack Engine, React, Tailwind CSS
- **Yapay Zeka:** Gemini 1.5 Pro & Gemini 2.5 Flash (Multimodal API)
- **Görsel/İşitsel:** Canvas API, KaTeX, Web Speech API (tr-TR)

## 🔑 Kurulum ve Kullanım

Uygulamayı tam kapasiteyle (Görüntü işleme ve otonom çözüm motoru dahil) kullanabilmek için kendi **Gemini API** anahtarınıza ihtiyacınız var.

### 1. Gemini API Key Nasıl Alınır?
1. [Google AI Studio](https://aistudio.google.com/) adresine gidin ve Google hesabınızla giriş yapın.
2. Sol menüden **"Get API key"** (API anahtarı al) butonuna tıklayın.
3. **"Create API key in new project"** seçeneğine tıklayarak yeni bir anahtar oluşturun.
4. Ekranda beliren `AIzaSy...` ile başlayan anahtarı kopyalayın. *(Not: API kullanımı standart sınırlar dahilinde ücretsizdir.)*

### 2. ARCONODE'da Nasıl Kullanılır?
1. [ARCONODE Canlı Demoyu](https://app-bgfanvos62v5.appmedo.com) tarayıcınızda açın.
2. Sol paneldeki **"GEMINI API KEY"** kutusuna kopyaladığınız anahtarı yapıştırın. *(Anahtarınız sadece tarayıcınızın lokal depolamasında saklanır, dışarıya aktarılmaz).*
3. Üstteki **"DEV BYPASS"** modunu kapatın.
4. Bir algoritma sorusu yazın veya bir fotoğraf (JPG/PNG) yükleyip **"ÇÖZÜMÜ BAŞLAT"** butonuna basın!
